# 分期生活 · AI 助手

面向日常消费与资金规划的轻量 Web 应用，将「分期」从表格与计算器里解放出来：用更接近真实决策流程的方式，理解每一期的负担，并在 AI 管家的辅助下完成方案推敲与复盘。

---

## 项目背景

在消费分期场景中，用户真正关心的往往不是「利率公式」，而是**每期还多少、总成本如何、方案是否可持续**。本项目以「分期生活」为产品叙事，把阶梯分期、方案对比与历史记录串联为一条完整路径；同时引入 **AI 助手**，在合规与可解释的前提下，帮助用户整理思路、补充建议与风险提示，让工具更像一位随身的财务参谋，而非冷冰冰的试算器。

---

## 核心亮点

- **iOS 风格 UI**  
  借鉴 iOS 的层次、圆角与动效节奏，在 Web 上呈现克制、清晰、可信赖的移动端体验，降低「金融工具」的距离感。

- **智能阶梯分期计算**  
  支持多段利率/期数逻辑下的试算与对比，突出「每期现金流」与「总成本」两条主线，便于快速判断方案是否匹配个人现金流。

- **localStorage 历史记录管理**  
  关键方案与试算结果在本地持久化，无需登录即可回溯；兼顾隐私与日常使用频率。

- **响应式手机预览模式**  
  桌面端亦可一键进入「手机画幅」预览，方便产品演示、截图与真实单手操作路径的验证。

## 环境要求

建议使用 **Node.js LTS**（如 20.x）与 **pnpm / npm / yarn** 其一作为包管理器。下文以 **npm** 为例。

---

## 安装

```bash
git clone <你的仓库地址>
cd my-project
npm install
```

---

## 运行

| 命令 | 说明 |
|------|------|
| `npm run dev` | 本地开发（Webpack） |
| `npm run dev:turbo` | 本地开发（Turbopack，启动更快） |
| `npm run build` | 生产构建 |
| `npm run start` | 启动生产服务（需先 `build`） |
| `npm run preview` | 构建并本地预览生产效果 |
| `npm run lint` | 代码检查 |

开发模式下默认访问：**http://localhost:3000**（以终端输出为准）。
